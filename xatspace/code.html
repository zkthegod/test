How to publish on GitHub Pages

  1. Create a GitHub repository. Name can be anything (e.g., my-xatspace).
  2. Click Add file → Upload files. Drag the three files from your chosen template: index.html, styles.css, script.js.
  3. Commit the changes.
  4. Open Settings → Pages. Under “Build and deployment”, select “Deploy from branch”. Branch: main. Folder: /root. Save.
  5. Wait 1–2 minutes for deployment. Your site will be available at the URL displayed in the Pages section.
  6. If you want different pages per template, create subfolders (e.g., neon/) and place each template’s 3 files in its own folder. Navigate to .../neon/ to open that page.
